# Emotion Wheel

An interactive visualization of the emotion wheel using HTML, CSS, and JavaScript with D3.js.

## Description

This project implements an interactive emotion wheel visualization that categorizes emotions into a hierarchical structure:

- Primary emotions (center): Happy, Surprised, Bad, Fearful, Angry, Sad
- Secondary emotions (middle ring): More specific derivatives of primary emotions
- Tertiary emotions (outer ring): The most specific emotion descriptions

## Features

- Visual representation of emotion relationships
- Color-coded sections for each primary emotion
- Interactive tooltips showing the full emotion path on hover
- Responsive design

## How to Run

1. Clone or download this repository
2. Open the `index.html` file in a web browser

No server or additional dependencies are required to run this locally.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- D3.js (data visualization library)
